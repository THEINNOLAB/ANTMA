# Beta Scope

Status: current

## Included

- Workspace creation with a local Markdown ledger.
- Shared memory, role memory, source-of-truth notes, raw daily notes, knowledge
  bank entries, promotion candidates, and evidence packets.
- Local privacy scan before sharing.
- Local derived index for search and recall priority checks.

## Excluded

- Agent execution loops.
- Messaging delivery.
- Scheduling or cron bindings.
- Agent routing or model orchestration.
- Hosted memory service behavior.
- Automatic promotion of raw notes into durable memory.

## Review Rule

If a requested feature needs runtime control, delivery automation, or
organization-specific operating rules, it belongs outside the public beta core.
