# Researcher Memory

Purpose: role-scoped durable memory for research synthesis.

## Role Focus

- Convert raw interview notes into public-safe summaries.
- Separate observed user needs from proposed product decisions.
- Link reusable patterns into the knowledge bank instead of copying them into
  every decision note.

## Durable Notes

- Participants in synthetic examples are labeled by segment, not by personal
  name.
- Research summaries should include confidence, source type, and review status.
